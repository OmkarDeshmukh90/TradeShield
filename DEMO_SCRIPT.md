# TradeShield 5-7 Minute Demo Script

## 0:00 - 0:45 | Setup and context
- Open `/` and briefly state the problem: fast-changing global disruption risk.
- Show Demo Mode badge and select scenario (`tariff` then `congestion`).

## 0:45 - 2:00 | Deterministic signal ingestion
- Enable Demo Mode and run ingestion.
- Point out recent events and scenario cards (tariff spike / congestion).
- Mention deterministic local replay: no external network dependency.

## 2:00 - 3:30 | Client-specific risk and recommendation
- Open top risk event and generate playbook.
- Explain 3 options (continuity-first, cost-balanced, margin-protect).
- Open “Why this recommendation” panel and walk factor bars + assumptions.

## 3:30 - 5:15 | Decision execution loop
- Approve first playbook step.
- Add a comment note.
- Update outcome to `monitoring` with ETA hours.
- Show that approvals/comments/outcomes are persisted and auditable.

## 5:15 - 6:30 | Operational credibility
- Open `/ops`.
- Show ingestion latency, alert status counts, and connector cooldown states.
- Mention correlation IDs in logs for traceability.

## 6:30 - 7:00 | Close
- Summarize value: from detection to explainable and trackable decision execution.
- Mention roadmap boundaries (what was intentionally deferred).
